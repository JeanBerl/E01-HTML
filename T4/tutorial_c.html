Estruturas de Repetição

Por quê?

Estruturas de repetição são utilizadas para repetir instruções até que uma condição se torne falsa/verdadeira

while

Segue o padrão while(condicao){} Executa as instruções enquanto a condição for verdadeira se na primeira vez for falsa ele não executa as instruções

Motivos para usar

  • Quando sua condição não tem um momento definido para se tornar falsa

Desvantagens

  • Caso você precise incrementar o valor da varíavel é comum esquecer e isso causa um loop infinito

for

Segue o padrão for(int i = 0; i < 10; i++){instruções} for(inicialização variavel; condição; incrementação)

Motivos para usar

  • A condição depende do valor de uma variável que é atualizada 1 vez em cada loop
  • Ajuda a evitar de formar loops infinitos

Desvantagens

  • Te obriga a incrementar o valor da variável, o que pode ser ruim caso você não precise disso

do while

Segue o padrão do{instruções}while(condicao);

Motivos para usar

  • Quando as suas instruções devem ser executadas ao menos 1 vez

Desvantagens

  • Seu loop sempre executa, o que pode ser indesejado

Exemplos

  • While

    int i = 0;
    while (i < 5){
    printf("aaa");
    i++;
    }

  • For

    for(int i = 0; i < 10; i++){
    printf("oi")
    }

  • Do While

    do{
    printf("alo")
    }while(1 == 2);
